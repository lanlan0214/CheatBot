# 桌面自動化工具（CheatBot）

這是一個 Windows 用的小工具，主要用途是：
- 自動左右移動（左按住 N 秒 → 右按住 N 秒）
- 長按期間可輪播技能鍵（例如 `K,1,2`）
- 一鍵檢查更新（有新版本就下載並重啟）

---

## 使用前準備

- 作業系統：Windows
- 建議用管理員身分執行（部分遊戲/程式需要）
- 目標視窗要先開啟

---

## 下載與啟動（一般使用者）

1. 到 GitHub 專案頁下載最新版本 `.exe`
2. 雙擊啟動 `桌面自動化工具.exe`
3. 若防毒或 SmartScreen 提示，請確認來源後允許執行

---

## 快速開始（3 步驟）

1. **選擇視窗**
	- 按「重新整理視窗」
	- 從下拉選單選你的遊戲/程式
	- 按「選擇視窗並切到前景」

2. **設定宏**
	- 長按秒數：例如 `5`
	- 技能鍵序列：例如 `K,1,2`
	- 技能間隔秒數：例如 `0.5`
	- 左右切換停頓秒數：預設 `0.05`（不穩可調到 `0.08~0.12`）

3. **開始執行**
	- 按「走路+放技能」開始
	- `F9`：暫停/繼續
	- 「停止」：中止執行

---

## 更新程式（使用者）

1. 開啟程式
2. 按「檢查更新」
3. 若有新版本，按確認
4. 程式會自動下載 → 關閉 → 更新 → 重啟

> 若你是用原始碼模式（不是 exe）啟動，會只開下載頁，不會自動覆蓋更新。

---

## 常見問題

### 1) 一直往左走 / 不會往右走
- 先把「左右切換停頓秒數」調高到 `0.08~0.12`
- 試「前景模式」（通常成功率最高）
- 以管理員身分啟動工具

### 2) 背景模式沒反應
- 有些遊戲不接受背景輸入（這是目標程式限制）
- 請改用前景模式

### 3) F9 沒反應
- 先點一下工具視窗再按 F9
- 若要全域熱鍵，建議用管理員身分啟動

### 4) 檢查更新失敗
- 確認網路可連到 GitHub
- 可能是暫時 API 連線問題，稍後重試

---

## 給維護者（簡短）

- 發新版前先更新 `src/app_config.py` 的 `APP_VERSION`
- 打包：

```bash
python -m PyInstaller "桌面自動化工具.spec"
```

- 到 GitHub 建立新 Release（tag 例如 `v1.0.1`）
- 上傳 `dist/桌面自動化工具.exe`
- 使用者端就能按「檢查更新」收到新版
